# 底材选择规则（全版本通用）

## 防具底材选择原则
1. 优先选择对应角色属性底材（力量/敏捷/智慧）
2. 终局底材等级要求：75 级以上
3. 隐含属性优先选择对 BD 有增益的（如攻速、暴击等）

## 武器底材选择原则
1. 匹配 BD 的伤害类型（物理/元素/混沌）
2. DPS 基础值是优先考虑项
3. 攻速底材适合叠加词缀 BD

## 首饰底材选择
- 戒指：优先生命 + 抗性底材
- 项链：优先暴击 / 施法速度底材（视 BD 而定）
- 手镯：优先全元素抗性底材

## 通用原则
- 不要在低级底材上浪费高级通货
- 优先用「偷看」确认词缀池再做装
