# POE2 全版本通用机制

## 做装基础流程
1. 确定 BD 词缀目标（主属性 + 辅助词缀）
2. 选合适底材（等级 / 底材类型）
3. 魔法装：蔚蓝 → 改造/扩充 → 如有需要镜铸
4. 稀有装：混沌洗 / 神圣强化 / 工艺台强行做装
5. 若需特定词缀，先封条再做

## 核心交易通货（玩家常用简称）
| 通货       | 作用                         | 外号           | 备注                     |
|-----------|------------------------------|----------------|--------------------------|
| 混沌石     | 重掷稀有装备所有词缀           | **C**          | 基础计价通货，最常用      |
| 神圣石     | 重置词缀数值范围               | **D**          | 硬流通货，高价值交易      |
| 崇高石     | 为稀有装备添加一条词缀          | **E** / EX     | 硬流通货，做装必备        |
| 卡兰德魔镜 | 复制一件装备                   | **镜子 / M**   | 最贵通货，毕业装复制      |
| 发辫       | 镜子装专用道具（0.3版本新增）   | **发辫**       | 百D起步，仅次于镜子       |
| 神圣预兆   | 突破物品等级区间上限到93级      | **预兆**       | 0.3版本新增，极高价值     |
